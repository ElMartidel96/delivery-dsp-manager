## Cómo Usar Estos Documentos

### Para el Equipo de Desarrollo
1. Lee ECOSISTEMA_INTEGRADOR.md para entender el TODO
2. Consulta antes de diseñar nuevas features
3. Mantén la visión en mente al escribir código

### Para el Equipo de Crecimiento/Ventas
1. Memoriza los pitches de PITCH_EJECUTIVO.md
2. Usa el MANIFIESTO para contenido de marketing
3. Refiere a las historias de éxito en conversaciones

### Para Liderazgo
1. Revisa trimestralmente para actualizaciones
2. Usa para alinear decisiones estratégicas
3. Comparte con nuevos socios/inversores

### Para Nuevos Miembros
1. Lee el MANIFIESTO primero (30 min)
2. Luego ECOSISTEMA (1 hora)
3. Finalmente PITCH para tus funciones específicas

---

## Principios de Actualización

1. **Testimonios**: Agregar nuevas historias de éxito mensualmente
2. **Métricas**: Actualizar números reales trimestralmente
3. **Estrategia**: Revisar roadmap cada 6 meses
4. **Pitches**: Refinar basado en feedback de campo
